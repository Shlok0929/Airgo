# AirGo - Drone Delivery Website

Welcome to **AirGo**, a modern drone delivery website built as a static HTML/CSS project. This repository contains the complete site files for the AirGo demo website, including the homepage, content pages, about page, contact page, and references.

## Overview

AirGo is a fictional drone delivery company website created to demonstrate a clean, responsive layout and navigation using HTML and CSS. The site includes branded page sections, an animated mobile menu button, a contact form, and social media links.

## Pages

- `Template/index.html` - Home page with a hero section and introductory content.
- `Template/content-01.html` - Content page for additional service or feature details.
- `Template/about.html` - About page describing the company's mission and vision.
- `Template/contact.html` - Contact page with a contact form and contact details.
- `Template/reference.html` - References page for citations or resource links.

## Features

- Responsive navigation bar with a hamburger menu for smaller screens.
- Clean hero layout with image and descriptive text.
- Company mission and vision section.
- Contact form UI for user submissions.
- Footer with social media icon links.
- Consistent styling across all pages.

## Technologies Used

- **HTML5** for page structure.
- **CSS3** for styling, layout, and responsive design.
- **Font Awesome** for iconography.

## Project Structure

- `Template/` - Contains all HTML page templates.
- `Style/style.css` - Main stylesheet used by every page.
- `Images/` - Contains image assets used by the website.
- `CSS Validator/` - Validation tools or reports for CSS.
- `HTML Validator/` - Validation tools or reports for HTML.

## How to Use

1. Open the `Template/index.html` file in a web browser.
2. Navigate between pages using the top menu.
3. Optionally, edit `Style/style.css` to customize colors, fonts, and layout.

## Notes

- The form on `Template/contact.html` is currently static and does not submit to a server.
- External icons are loaded from Font Awesome via CDN.
- The site is designed as a simple static prototype and can be extended with JavaScript or backend integration.
